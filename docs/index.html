About me

Okay, so I don't really dress up in armour, but when it comes to tackling your web-related needs - I do have quite an armoury. My name is Mark (I go by my middle name) and I'm a freelance web developer with a focus on Javascript and now React.js. I'm into modular, re-usable code that's version controlled to make responsive sites. I have a background in System Applications analysis, Logistics and banking. I'm experienced with high pressure “fix it now” environments, where attention to detail is crucial. I'm an equal left/right brain thinker - creative enough to design while practical enough to code. Making cool stuff on the web is a natural progression for me.

Some contents of my toolbox

This isn't an exhaustive list but it gives some idea of what i've worked or am working with. I'm neither an expert, nor a beginner - I'm somewhere in-between. I just keep trucking on, learning more so that every day I get an "ah ha" moment. If what you see below doesn't suit the project you have - Lets talk about it. I'm always open to learning new things.

HTML/CSS/Javascript

HTML5 for markup, CSS3 for styling and Javascript or object-orientated inter-activity. Just starting to get my head around ES6 with Babel.js

Gulp.js

The build tool that’s increasing in popularity due to it’s speed and ease at automating the boring stuff.

Sass and PostCSS

Sass is the most popular pre-processor. PostCSS - the new post-processor is the fastest to compile of all pre and post-processors. Both allow a more modular, organized and maintainable CSS build.

Node.js and NPM

Node.js - The platform built by Google that introduced Javascript to the server and exploded into the largest ecosystem of open source libraries in the world with NPM - the Node Package Manager

Git and Github

Git - the most popular version control system for project history tracking and collaboration through feature branches.Github is the repository hosting service used with each git project.

React.js and Meteor

React.js is the open source Javascript library made by the FaceBook team for building User Interfaces. Meteor is the full-stack javascript platform for making Angular2 and React projects easier. It comes with well organized build tools and its own version of Meteor specific NPM called Atmosphere.js.
